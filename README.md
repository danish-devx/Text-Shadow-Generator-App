## 📌 Project Overview
Text Shadow Generator App is a lightweight and interactive frontend tool built with HTML, CSS, and JavaScript. It enables users to dynamically generate and preview CSS `text-shadow` properties in real time.

This project demonstrates practical understanding of DOM manipulation, event handling, and dynamic styling.

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🔥 Key Features
- Real-time text-shadow preview
- Adjustable horizontal offset
- Adjustable vertical offset
- Blur radius control
- Custom shadow color selection
- Instant CSS output generation
- Responsive and clean UI
